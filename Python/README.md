## iPython Notebooks

Contains iPython Notebook files for Python 3.5+ basics from the course [Python for Data Science](https://courses.edx.org/courses/course-v1:Microsoft+DAT208x+6T2016/info) in EDX.org.

:books: **Syllabus:** 

- Module 1: Python Basics
- Module 2: Python Lists
- Module 3: Functions and Packages
- Module 4: Numpy

Write superfast code with Numerical Python, a package to efficiently store and do 
calculations with huge amounts of data.

- Module 5: Matplotlib

Create different types of visualizations depending on the message you want to convey. 
Learn how to build complex and customized plots based on real data.

- Module 6: Control flow and Pandas

Write conditional constructs to tweak the execution of your scripts and get to know 
the Pandas DataFrame: the key data structure for Data Science in Python.
