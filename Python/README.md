## Python 3 projects:
#### :octocat: **Epubify:** 
Convert Articles to .epub format and automatically save them to Dropbox. 
The idea was to be able to read articles which I'd normally save in "Pocket" on my regular
ebook reader. 

__Using packages:__
- dropbox
- pocket
- BeautifulSoup

#### :octocat: **Twitter spark app:** 
PySpark application, fully OOP, using Twitter data (in dutch), to be reworked with recent twitter data ini English, soon.
OOP project, which fetches information from the tweets and can output trends in twitter between topics. 
Can be extended in order to do other cool stuff from twitter data like ML and statistics 

__Using packages:__
- pySpark
- sparkML
- tweepy (for automatic fetching of data) - the complete implementation is not working yet.

#### :octocat: **Youtuber:** 
Application whose intent is to get a URL of a youtube playlist and then download all the tracks from there as MP3 format, with highes possible quality available, in order for the user to listen offline.
